# Physical AI & Humanoid Robots – An AI-Native Book

This repository contains a comprehensive AI-native textbook that teaches Physical AI and Humanoid Robotics by bridging artificial intelligence.

## Mission
Create a comprehensive AI-native textbook that teaches Physical AI and Humanoid Robotics by bridging artificial intelligence.

## Structure
The textbook is organized in 4 parts with 20 cohesive chapters:

### Part 1: Foundations
1. Introduction to Physical AI
2. Fundamentals of Humanoid Robotics
3. Sensors and Perception Systems
4. Actuators and Mobility Systems
5. Introduction to AI for Robotics

### Part 2: Perception
6. Computer Vision for Robotics
7. Spatial Reasoning and Mapping
8. Tactile and Proprioceptive Sensing
9. Auditory Perception
10. Sensor Fusion and State Estimation

### Part 3: Control
11. Locomotion and Balance Control
12. Manipulation and Grasping
13. Planning and Decision Making
14. Learning and Adaptation
15. Human-Robot Interaction

### Part 4: Applications
16. Cognitive Architectures
17. Advanced Control Strategies
18. Robotics Software Architectures
19. Applications and Case Studies
20. Future Directions and Challenges

## Setup and Development

### Prerequisites
- Node.js (version 18 or higher)

### Installation
```bash
npm install
```

### Running the Development Server
```bash
npm start
```
The site will be available at http://localhost:3000/

### Building for Production
```bash
npm run build
```

### Deployment
The site can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting service.
